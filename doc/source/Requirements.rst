=============
Requirements
=============

**DiscTransformation**

DiscTransformation requires the following external packages.

1. Numpy (Harris et. al., Nature, 2020)
	Install it using conda package manager (recommended)

2. Scipy (Virtanen et. al., Nature Methods, 2020)
	Install it using conda package manager (recommended)
